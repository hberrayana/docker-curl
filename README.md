# docker-curl
Alpine-based image with just curl. It has also the core-utils package installed on top of it

## Usage

```console
$ docker run --rm appropriate/curl -fsSL https://www.google.com/
```

## Tags

* `appropriate/curl:latest`: based on `alpine:latest`
